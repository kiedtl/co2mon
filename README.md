# Carbon Dioxide Monitor

Hardware:

- SCD40 Sensirion sensor.
- Raspberry Pico. AliExpress knockoff works too.
- 128x64 SCD1306 OLED display.
- nRF24L01+PA+LNA radio module.

Software:

- Pico C SDK.
- RF24(-Network/-Mesh) libraries.
- u8g2 OLED display driver.
